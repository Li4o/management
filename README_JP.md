# Flexible Inventory Manager
> **あらゆるタイプの備品を管理するために設計された汎用性の高い管理アプリです。**
> **ユーザーは自分の決めた項目でカスタマイズすることができます。**

## 技術選定
[Frontend] React (TypeScript) + Vite + Tailwind CSS (or Shadcn UI)
[Backend]  Node.js (Express or NestJS) with TypeScript
[Database] PostgreSQL (with Prisma ORM)
[Docs/UI] Markdown / OpenAPI (Swagger)
